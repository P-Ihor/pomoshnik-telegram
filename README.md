# Помощник (Pomoshnik) - Telegram AI Assistant

Личный AI-ассистент для Telegram, вдохновленный J.A.R.V.I.S. 
Базируется на мощном фреймворке автономных агентов **OpenClaw** и использует кастомный **Python MCP Server** для расширения возможностей (анализ фото, видео, документов, извлечение дат, поиск по геолокации).

## Архитектура

Проект оптимизирован для работы на бесплатных серверах (например, Oracle Cloud Free Tier x86_64, 1 OCPU, 1 GB RAM).

- **OpenClaw (Node.js)**: Мозг бота. Интеграция с Telegram, память (SQLite), напоминания (cron), обращение к моделям через OpenRouter.
- **Python MCP Backend**: "Руки" бота. Выполняет специализированные задачи, которые требует ML-библиотек или сложных парсеров (Vision, Whisper, OCR, QR, YT-DLP).

## Поддерживаемые типы сообщений
- 💬 **Текст**: Свободное общение, поиск по памяти, планирование задач.
- 📸 **Фото**: Глубокий анализ (Gemini Vision) + распознавание QR/Barcode.
- 🎬 **Видео/Video Note**: Распознавание происходящего на видео.
- 🎙️ **Аудио/Голосовые**: Транскрипция (Whisper/Gemini) и понимание намерений.
- 📄 **Документы**: Извлечение текста (PDF) и анализ.
- 📍 **Локация**: Конвертация в адрес (Reverse Geocoding).
- 🔗 **Ссылки**: Извлечение статей (Trafilatura) и видео (yt-dlp).

## Быстрый старт (Локально)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/pomoshnik-telegram.git
   cd pomoshnik-telegram
   ```

2. Скопируйте файл конфигурации и заполните токены:
   ```bash
   cp .env.example .env
   ```

3. Запустите через Docker Compose:
   ```bash
   docker-compose up -d
   ```

## Развертывание на Oracle Cloud (Free Tier)
Для автоматической настройки Ubuntu VM используйте скрипт:
```bash
bash scripts/setup-oracle.sh
```

## Multi-user масштабирование
Текущая конфигурация предназначена для одного пользователя (владельца). Для добавления других пользователей, необходимо продублировать сервис `pomoshnik` в `docker-compose.yml`, создав независимые тома данных (volumes) для каждого пользователя и указав им разные токены Telegram-ботов. Python MCP сервер может быть один на всех.
